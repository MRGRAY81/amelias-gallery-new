# 🚀 Deploy Amelia's Gallery to Replit

## Quick Start Guide

### Step 1: Push to GitHub
1. Click **"Save to GitHub"** button in Emergent
2. Create a new repository (e.g., "amelias-gallery")
3. Push all files to GitHub

### Step 2: Import to Replit
1. Go to [Replit.com](https://replit.com)
2. Click **"Create Repl"**
3. Choose **"Import from GitHub"**
4. Select your repository
5. Click **"Import from GitHub"**

### Step 3: Run on Replit
Replit will automatically:
- Detect the Node.js project
- Install dependencies (`npm install`)
- Start the server (`node index.js`)
- Provide a live URL

### Step 4: Access Your Gallery
Once running, Replit will give you a URL like:
`https://amelias-gallery.your-username.repl.co`
