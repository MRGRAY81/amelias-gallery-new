- server/          → Node.js + Express backend
- public/          → HTML/CSS/JS frontend  
- README.md        → Full documentation
- PROJECT_SUMMARY.md → Complete feature list
- QUICK_REFERENCE.md → Quick access guide
- start-server.sh  → Startup script

## 🚀 Quick Start (3 Steps)

### Step 1: Extract the ZIP
Unzip amelias-gallery.zip to your desired location

### Step 2: Install Dependencies
```bash
cd server
npm install
```

### Step 3: Start the Server
```bash
node index.js
```

Server will run on: http://localhost:4000

## 🌐 Access the Website
- Home: http://localhost:4000/
- Gallery: http://localhost:4000/gallery.html
- About: http://localhost:4000/about.html
- Contact: http://localhost:4000/contact.html
- Admin: http://localhost:4000/admin/login.html

## 📝 Important Notes
- This is a Multi-Page Application (MPA) with standard HTML files
- NO React, NO SPA routing - just pure HTML/CSS/JS
- Backend uses Node.js + Express
- Database is a simple JSON file (server/db.json)
- Port 4000 is configurable via process.env.PORT
